# Code of Conduct

## Our Standard

This project is intended to be a respectful, professional, and growth-focused space.

By participating, you agree to:

- Treat everyone with respect and courtesy.
- Give feedback that is specific, constructive, and kind.
- Assume good intent while still communicating clearly.
- Stay open to differing technical perspectives.
- Keep discussions relevant to improving the project.

## Unacceptable Behavior

The following behavior is not accepted:

- Harassment, discrimination, or personal attacks.
- Trolling, hostile language, or intimidation.
- Publishing private information without consent.
- Repeatedly derailing discussions or spam behavior.
- Plagiarism or misrepresenting others' work as your own.

## Scope

This Code of Conduct applies to all project spaces, including:

- Issues
- Pull requests
- Discussions
- Any communication channels linked to this repository

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that violate this Code of Conduct.

Repeated or severe violations may result in temporary or permanent restrictions from participating.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the repository owner/maintainer.

All reports will be reviewed fairly and handled with discretion.
